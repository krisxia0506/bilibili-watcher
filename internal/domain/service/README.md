# Domain Services

此目录包含领域服务（Domain Services）的定义。

## 主要职责

*   封装那些不适合放在任何单个实体或值对象中的领域逻辑。
*   处理涉及多个领域对象交互的复杂业务操作。
*   执行显著的业务流程计算。
*   确保领域模型（实体和值对象）保持内聚，避免承担过多的职责。
*   **不**包含任何基础设施相关的代码（如数据库访问、API 调用），如果需要数据，通过注入的仓库接口获取。
*   通常是无状态的。
